Develop

.. meta::
   :description: Author, run, and iterate on Lakehouse Plumber pipelines locally — the web IDE, the in-IDE AI assistant, sandboxes, editor autocomplete, and coding agents.

How you write and iterate on flowgroups is up to you — a text editor and the CLI,
a browser IDE, or a coding agent. These guides cover each authoring surface.

- :doc:`web-ide` — a browser UI with a graph view and live generation.
- :doc:`ai-assistant` — authoring help built into the web IDE.
- :doc:`sandbox` — an isolated, scoped copy of your pipelines for safe iteration.
- :doc:`editor-setup` — YAML autocomplete and validation in your editor.
- :doc:`coding-agents` — drive Lakehouse Plumber from an AI coding agent.
