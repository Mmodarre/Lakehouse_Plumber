The full course
===============

You'll work through:

1. **Install and initialize** — get the ``lhp`` command and scaffold a project.
2. **Your first pipeline** — ingest CSV into a bronze streaming table.
3. **Add a transform and data quality** — reshape rows and quarantine bad ones.
4. **Reuse with a preset** — factor shared config out of your flowgroups.
5. **Templates** — stamp out many tables from one pattern.
6. **Scale with blueprints** — generate whole pipelines across environments.
7. **Deploy with bundles** — ship to a Databricks workspace.

The course is complete end to end — every step is fixture-backed and generates
the code shown.
